Codemax CLI

Autonomous coding CLI with chat, TUI, web mode, and OpenAI-compatible local API.

GitHub Repository Deployment Docs

NPM Installation

Install project dependencies and build the CLI:

npm install && cd cli && npm install && npm run build && cd ..

Deploy

Run the one-command deploy/setup flow:

npm run deploy

Codemax NPM Commands

Use these updated command aliases from the project root:

npm run codemax
npm run codemax:install
npm run codemax:dev
npm run codemax:build

CLI Modes

node cli/dist/cli.js chat
node cli/dist/cli.js tui
node cli/dist/cli.js start

OpenAI-Compatible API

Codemax via Ollama OpenAI endpoint:

curl -sS http://127.0.0.1:11434/v1/chat/completions \
  -H "Content-Type: application/json" \
  -d '{
    "model": "codemax:latest",
    "messages": [
      { "role": "user", "content": "Generate a Node.js hello world API." }
    ]
  }'
